## 계층별 모델 선택 근거

### L1 OCR: Upstage Document Parse Enhanced
- 한국어 수식/표/그래프 인식 최고 수준
- KoNET 벤치마크에서 OCR 품질이 결정적 병목으로 확인
- Enhanced Mode: $0.02/page (수식/차트 특화)
- **대안**: GPT-4o Vision (보완 OCR로 병행 가능)

### L2 분류/검산: GPT-4o-mini
- JSON Structured Output 100% 스키마 준수 보장
- $0.15/1M input tokens (초저비용)
- 분류/검산 등 구조화 작업에 최적
- **대안**: Gemini Flash (유사 비용, 성능)

### L3 해설/오답분석: Claude Sonnet
- STEM 추론 최고 성능 (Chain-of-Thought 품질)
- 단계별 수학 풀이 해설 품질 우수
- $3/1M input tokens
- **대안**: GPT-4o (수학 점수 더 높으나 비용도 높음)

### L4 변형문항 생성: Claude Sonnet
- 고추론 작업(교정 의도 반영 문항 생성)에 강점
- 교육적 맥락 이해 우수
- 오답 문항만 처리하므로 비용 제어 가능

## KoNET 벤치마크 시사점 (NAACL 2025)

1. 검정고시 90%+ → 수능 50-60%로 급락: 난이도 높은 문제일수록 모델 선택 중요
2. OCR 품질이 한국어 시험 풀이의 결정적 병목: Upstage OCR 선택 근거
3. LLM + OCR 파이프라인 > 순수 MLLM: 계층적 모델 운영(L1~L4) 전략 지지
4. 모델 크기 ≠ 성능: 비용효율적 모델 선택 가능 (GPT-4o-mini의 가성비)
